Task1_Data-Analyst
Excel Sales Dashboard
Project Overview
This project focuses on analyzing sales data and creating an interactive dashboard using Microsoft Excel. The dashboard helps users understand business performance through visual reports and key performance indicators.

Objectives
Analyze sales performance using Excel.
Visualize Total Sales, Profit, and Sales Trends.
Identify high-performing and low-performing categories.
Present business insights through an interactive dashboard.
Tools Used
Microsoft Excel
Pivot Tables
Pivot Charts
Conditional Formatting
Data Cleaning Techniques
Project Workflow
1. Data Cleaning
Removed duplicate records.
Checked for missing or incorrect values.
Organized data into a structured format for analysis.
2. Data Analysis
Calculated Total Sales and Total Profit.
Analyzed category-wise performance.
Evaluated sales trends over time.
3. Dashboard Creation
Created Pivot Tables for data summarization.
Designed interactive charts for visualization.
Added slicers and filters for easy navigation.
Applied conditional formatting to highlight performance.
Key Insights
Identified top-performing product categories.
Analyzed profit contribution across categories.
Observed sales growth and decline trends.
Highlighted areas requiring business improvement.
Skills Demonstrated
Data Cleaning
Data Analysis
Microsoft Excel
Pivot Tables
Data Visualization
Dashboard Design
Business Analytics
Conclusion
This project demonstrates how Excel can be used to transform raw sales data into meaningful insights through data analysis and interactive dashboard design.
